# 萬代幣計算機

萬代幣計算機是一個旨在計算各種萬代幣的組合並減少浪費的專案。使用者可以輸入目標金額，程式會自動計算出最接近目標金額的萬代幣組合，並顯示所需的最少萬代幣數量。這個專案可以幫助使用者更有效地管理萬代幣，減少浪費和不必要的支出。

## 使用方法

點擊下方儲存格右上執行按鈕執行程式碼，在下方輸入框填入目標金額後按下Enter鍵。計算機將自動算出最接近目標金額的萬代幣組合，並顯示所需的最少萬代幣數量。
